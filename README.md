# soil-mechanics
Tools for teaching soil mechanics

Development:
 - `npm install`
 - `npm run dev`

Production build:
 - `npm run build`
 - `npm run preview`

Available tools:
- [Soil compaction](https://scigem.github.io/soil-mechanics/compaction.html)
- [Darcy flow](https://scigem.github.io/soil-mechanics/darcy-flow.html)
- [Mohr's circle and failure criteria](https://scigem.github.io/soil-mechanics/mohrs-circle.html)
- [Void ratio explainer](https://scigem.github.io/soil-mechanics/ruler.html)
- [Sieve analysis grading curve](https://scigem.github.io/soil-mechanics/sieve-analysis.html)
- [Elastic strip footing displacement field](https://scigem.github.io/soil-mechanics/elastic-footing.html)
- [1D consolidation explorer](https://scigem.github.io/soil-mechanics/consolidation.html)
- [1D compression explorer](https://scigem.github.io/soil-mechanics/1d-compression.html)
- [Newmark's chart](https://scigem.github.io/soil-mechanics/newmarks-chart.html)
- [System dynamics](https://scigem.github.io/soil-mechanics/system-dynamics.html)
- [Critical state line visualisation](https://scigem.github.io/soil-mechanics/critical-state.html)
